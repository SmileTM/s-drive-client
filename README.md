## ✨ 核心特性 (Features)

S Drive Client 是一个集高颜值与高安全性于一体的跨平台文件管理工具，专为管理私有存储而生。

### 🎨 极致 UI 设计
*   **浮岛式视觉系统**：侧边栏采用大圆角悬浮设计，配合毛玻璃特效与流畅的 Framer Motion 动画。
*   **灵动操作栏 (Dynamic Action Bar)**：底部功能区随选中状态智能交互，重命名、删除、移动一触即达。
*   **状态实时监测**：云盘离线状态自动感应，侧边栏视觉提醒，确保连接始终可见。
*   **全平台响应**：
    *   **桌面端**：高效多列布局 + 拖拽上传下载。
    *   **移动端**：原生级沉浸式体验，支持触控手势与社交分享接收。

### 🚀 强大的协议支持与播放引擎
*   **双协议聚合**：高度优化的 **WebDAV** 与 **SMB/CIFS (Windows 共享)** 原生直连支持。
*   **Hyper-Turbo 流式播放**：
    *   内置万能播放器，支持视频/音频秒开、任意进度拖拽。
    *   深度优化的缓存机制与资源回收，告别卡顿与“僵尸”进程。
*   **原生性能加持**：Android/iOS 端采用完全重写的 Native 插件，彻底解决移动端网络限速与权限问题。

### 🛡️ 安全与隐私
*   **零信任存储**：所有敏感凭据在本地经过 **AES-256** 加密存储。
*   **隐私优先**：完全本地化运行，所有数据在用户设备与服务器间点对点传输。
